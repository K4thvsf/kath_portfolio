About Me

Bridging Biology and Technology

I'm a passionate Bioinformatician and Data Scientist with an MSc in Molecular Biology (Bioinformatics) and a BSc in Biomedicine. Currently based in Padua, Italy, I specialize in developing computational biology pipelines and machine learning models that unlock insights from complex biological datasets.

My expertise spans across Python, R, SQL, and high-performance computing, with a particular focus on AI and deep learning applications in biological research. I co-created PhageSeeker, a Python script for detecting bacteriophage infections in bacterial genome assemblies, demonstrating my commitment to developing practical bioinformatics tools.

When I'm not analyzing genomic data or training neural networks, you'll find me reading science fiction, playing piano, traveling to new places, or curating the perfect indie music playlist. I believe that diverse interests fuel creativity in scientific research.

Bioinformatics • Data Analysis

PhageSeeker

Bioinformatics tool for detecting and analyzing bacteriophage sequences in bacterial genome assemblies. Uses BLAST database searches and pandas for data processing to identify viral sequences in FASTA format files.

Python pandas BLAST Database Sequence Analysis FASTA Analysis Data Processing

Methods: BLAST database searches for sequence alignment, pandas for data manipulation and analysis, and comprehensive reporting of identified viral sequences in genomic datasets.

Data Analysis • Statistical Modeling

TCGA Glioma Data Analysis

Statistical analysis of 1,122 brain glioma samples from TCGA using R and Bioconductor. Data cleaning through gene set analysis, heatmap clustering, and differential expression analysis.

R Bioconductor omicsdata Statistical Analysis Data Visualization Exploratory Analysis Data Cleaning

Methods: RNA expression analysis, DNA methylation profiling, molecular subtype classification, and statistical analysis of clinical variables including age and ethnicity impacts on patient outcomes.

Bioinformatics • Data Science

ORF Prediction Tool

Python algorithm for predicting open reading frames (ORFs) in nucleotide sequences. Implements start codon detection, stop codon identification, and genetic code translation with BRCA2 gene analysis example.

Python Bioinformatics Tool Development Sequence Analysis Genetic Code String Processing

Methods: ATG start codon detection, stop codon identification (UAA, UAG, UGA), and genetic code translation with error handling.

Machine Learning • Optimization

Genetic Algorithm Implementation

Machine learning implementation using genetic algorithms for optimization and predictive modeling. Features statistical analysis, fitness evaluation metrics, data-driven selection algorithms, and model performance optimization across 30+ generations.

Python NumPy Machine Learning Optimization Predictive Modeling Statistical Analysis

Methods: Population generation, fitness evaluation, selection algorithms, crossover breeding, and mutation operations for evolutionary optimization.
